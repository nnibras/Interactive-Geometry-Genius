Questions and Answers

What is DOM and what is the purpose of DOM?

DOM stands for Document Object Model. It is used for dynamically accessing, updating, adding and deleting the contents, styles and so on of scripts and programs so that it can meet the requirement of the application or website.

How to select elements using DOM? Name the DOM methods.

Elements from DOM can be selected using 5 different selectors.
The selectors are as follows:
1- getElementsByTagName() which uses tag name to access the element.
2- getElementsByClassName() which uses class name to access the element.
3- getElementById() which uses id name to access the element.
4- querySelector() which selects the first element of the specified CSS selector.
5- querySelectorAll() which selects all the elements of the specified CSS selector.

What is Event Bubble?

Event Bubble is a concept in the DOM. It takes place when an element receives the event parameter. This is when the DOM tree is traversed. At first the whole tree is traversed until the target node is found, then from the target node the event is bubbled to the top of the tree until it reaches the first parent node.

What is callback function and when should it be used?

The concept of callback function is such that, a function is passed as a parameter of another function, so that it invokes the outer most function to complete any action written on the function. It is mainly used to complete some kind of routine or action.
